# ToDoList

Simple React Todolist Application

## Features

- Create a new todo
- Update a todo (`'DONE'` or `PENDING`)
- Delete a todo

## How to run locally

1. Clone the repository
2. `npm install`
3. `npm run dev`

> **NOTE**: Currently, dummy data is included in the project, so it is ready to run without any DB!
